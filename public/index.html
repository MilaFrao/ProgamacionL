Cómo escribir un problema

Debes cumplir una sintaxis estándar para resolver tu problema de forma clara.

Paso 1

Define la función objetivo con max o min.

Ejemplo: max = 3x1 + 2x2

Paso 2

Escribe las restricciones, una por línea.

Ejemplo: 2x1 + 1x2 <= 18

Ejemplos

max = 1x1 + 2x2
1x1 + 3x2 >= 11
2x1 + 1x2 >= 9

Notas

  • No dejes espacios entre el coeficiente y la variable.
  • Usa operadores <=, >= o =.
  • Para Simplex puedes usar variables del tipo x1, x2, x3, etc.

Errores comunes

  • Escribir varias variables en una misma línea sin separar bien.
  • Usar un operador distinto a los permitidos.
  • Ingresar variables con subíndice distinto a 1 y 2 en el método gráfico.
