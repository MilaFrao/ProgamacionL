Escribe las restricciones

Cada restricción en una línea nueva

2x1 + x2 <= 18 x1 + 3x2 >= 24 3x1 + 2x2 = 36 x1 - 2x3 <= 100
¿Más especificaciones?
  • Coeficientes: números reales
  • Variables: x1, x2, ..., xn (simplex) / x1, x2 (gráfico)
  • Operadores: <=, >=, =
  • Simplex agrega automáticamente variables artificiales (R) y de holgura (s)

📋 Especificaciones detalladas:

  • El coeficiente debe ser un número real.
  • Las variables deben ser identificadas con x1, x2 para el método gráfico.
  • Las variables en el método simplex pueden ir de x1 hasta xn.
  • Los operadores permitidos son <=, >= y =.
  • Simplex agrega variable artificial (R) y variable de holgura (s).
