Programacion lineal

MetodoSimplex

Programa para la resolucion de problemas de programacion lineal mediante el metodo simplex dos fases o grafico.



¿Como usarlo? ¡Facil!

Debes cumplir una sintaxis estandar para resolver tu problema...¡sencillo sin problemas!

Paso 1
Para empezar, empieza por definir la funcion objetivo (Min/Max) Por ejemplo : max = 3x1 + 2x2 + ... + Axn
¡NOTA! que no pueden haber espacio entre el coeficiente y la variable, ademas de haber un espacio entre los terminos y los signos operacionales

Paso 2
Acontinuacion, escribe las restricciones a las que se sujeta el problema, ¡diferencialas con un salto de linea!

Por ejemplo :

  • 2x1 + 1x2 <= 18
  • 2x1 + 3x2 <= 42
  • 3x1 + 1x2 >= 24

Para simplex:

  • x1 + x2 +x3 <= 500
  • 5x1 + x2 -x3>= 200
  • x1 + 3x2=240
  • x1 -20x3<=0
  • x3<=6000

De igual manera no te preocupes, esta un ejemplo para ti en el cuadro de texto.
¿Mas dudas de como usarlo?, aqui una lista de mas especificaciones de la sintaxis.

  • El coeficiente debe ser un número real.
  • Las variables deben ser identificadas con x1, x2 para el metodo grafico etc.
  • Las variables en el metodo simplex si pueden ir del x1 hasta xn.
  • Los operadores permitidos son <=, >= y =.
  • Especificaciones del metodo simplex: se agrega una variable artificial (R), variable de holgura (s)
