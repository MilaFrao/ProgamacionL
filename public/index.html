¿Cómo usar esta herramienta?

Paso 1: Define la función objetivo

Escribe la función que deseas maximizar o minimizar en la forma:

max = 3x1 + 2x2 + ... + Axn

Nota importante: No debe haber espacios entre el coeficiente y la variable. Debe haber espacios entre los términos y los signos operacionales.

Paso 2: Escribe las restricciones

A continuación, escribe las restricciones separadas por saltos de línea. Cada restricción debe incluir operadores válidos (≤, ≥, =):

2x1 + 1x2 <= 100 x1 + 3x2 = 200 x1 + 0.5x2 >= 150
Especificaciones técnicas:
  • Coeficientes: Deben ser números reales (enteros, decimales o fracciones)
  • Variables en Método Gráfico: Solo se permiten x1 y x2
  • Variables en Método Simplex: Se permiten x1, x2, x3, ..., hasta xn
  • Operadores permitidos: <= (menor o igual), >= (mayor o igual), = (igual)
  • Variables de holgura: El simplex automáticamente agrega variables de holgura (s) y artificiales (R) según sea necesario
Interpretación de resultados:
  • Solución Óptima: Los valores de las variables (x1, x2, ..., xn) que optimizan el objetivo
  • Valor Z: El valor óptimo de la función objetivo
  • Iteraciones: Número de pasos realizados por el algoritmo simplex
  • Estado: Indica si la solución es factible, óptima o si hay restricciones no satisfechas
  • Tablas Simplex: Muestra el progreso del algoritmo en cada iteración
